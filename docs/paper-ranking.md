# Paper ranking

중요도를 Impact Factor 하나로 판단하지 않습니다.

현재 설명 가능한 점수:

- 검색어 관련도 45%
- 최신성 20%
- 인용 수 20% (100회에서 상한)
- DOI·초록·저자·저널 데이터 완전성 15%

구성 요소는 `ranking_components_json`과 화면에 표시됩니다. Crossref 인용 수는 Crossref 출처이며 다른 DB 값과 같다고 가정하지 않습니다. OA와 저널 지표는 해당 provider를 추가하기 전에는 미상입니다.

요약은 핵심/이야기형/엔지니어/실무 적용/비판적 검토 모드를 제공합니다. 이야기형도 과학적 흐름만 자연스럽게 연결하며 동화·캐릭터·억지 대화를 사용하지 않습니다. 초록만 있으면 `abstract_only`, 없으면 `metadata_only`를 표시합니다.
